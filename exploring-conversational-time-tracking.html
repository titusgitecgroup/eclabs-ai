Exploring Conversational Time Tracking through Whisper, spaCy and Mirascope

Our objective is to be able to extract specific entity information from audio file or text using Whisper, spaCy and Mirascope.io.

Quick Intro

Our objective is to be able to extract specific entity information from audio file or text using Whisper, spaCy and Mirascope.io.
  1. Whisper:
    It is an automatic speech recognition (ASR) system developed by OpenAI. It is designed to transcribe spoken language into written text with high accuracy
  2. Mirascope.io:
    It is a platform that leverages the power of OpenAI's Whisper models to provide transcription and voice-to-text services and here it is used to extract information for TaskDetails entity from the text.
  3. spaCy:
    It is an open-source library for Natural Language Processing (NLP) in Python.

Pre-requisites

The following prerequisites needed to be installed or created before getting into it
  • Python 3.9x version
  • A valid OpenAI API key
  • Mirascope
  • spaCy

TaskExtractor
  • TaskExtractor class is designed to create a prompt for extracting task entities from a text conversation using a specified prompt template. It inherits from a generic OpenAIExtractor class and utilizes a prompt template with placeholders for task details and the current date. The class attributes include the task text and the current date in ISO format.
    • Class Definition
      • Class Inheritance
        • TaskExtractor inherits from OpenAIExtractor[T]: This implies that OpenAIExtractor is a generic class, and TaskExtractor is a specialized version of it, likely tailored for extracting task entities.
      • Class Attributes
        • prompt_template:
          • A string attribute that stores the description of the task, its default value is an empty string.
          • It includes placeholders {task} and {today}.
          • The prompt instructs to convert the provided text conversation into task entities and sets the task's end date based on specific keywords.
        • task:
          • A string attribute intended to store the text conversation from which tasks will be extracted.
        • today:
          • A string attribute that stores the current date in ISO format.
          • It uses the function dateInISOFormat(today()), which converts the current date to an ISO string format.
      • Explanation
        • dateInISOFormat and today Functions
          • dateInISOFormat(date): Converts a date object to an ISO format string.
          • today(): Returns the current date and time.

split_tasks_using_spacy
  • split_tasks_using_spacy function takes a text input and a list of action verbs. It uses spaCy, a natural language processing library, to split the text into individual tasks based on the presence of these action verbs in the sentences.
  • Function Definition def split_tasks_using_spacy(txt: str, action_verbs: List[str]) - Defines the function which takes two parameters:
    • txt: A string containing the input text.
    • action_verbs: A list of action verbs used to identify task-related sentences.
  • Load spaCy Model
    • nlp = spacy.load("en_core_web_sm") - Loads the English core small model from spaCy, which contains pre-trained pipelines for various NLP tasks.
  • Process the Text
    • doc = nlp(txt) - Uses the spaCy model to process the input text, creating a Doc object that contains linguistic annotations.
  • Initialize Tasks List
    • tasks: list[str] = [] - Initializes an empty list to store the extracted tasks.
  • for each Iteration
    • Iterate Over Sentences:
      for sent in doc.sents: - Iterates over the sentences in the processed text (Doc object).
      Process Each Sentence:
      sent_doc = nlp(sent.text.strip()) - Processes each sentence individually.
      Check for Action Verbs:
      for token in sent_doc: - Iterates over the tokens (words) in the sentence.
      if action_verbs: - Checks if the list of action verbs is provided.
      if token.lemma_ in action_verbs: - Checks if the lemma (base form) of the token is in the list of action verbs.
      tasks.append(sent.text.strip()) - If an action verb is found, adds the sentence to the tasks list.
      break - Breaks out of the loop once an action verb is found in the sentence.
      else:
      - If no action verbs are provided, adds all sentences to the tasks list.
    • Return the Tasks List
      return tasks - Returns the list of extracted tasks.
from typing import List import spacy # Load the spaCy model nlp = spacy.load("en_core_web_sm") def split_tasks_using_spacy(txt:str, action_verbs:List[str]): try: tasks:list[str] = [] doc = nlp(txt) # Extract tasks for sent in doc.sents: # Parse the sentence sent_doc = nlp(sent.text.strip()) for token in sent_doc: # Check if the token is an actionable verb and the root of the sentence if action_verbs: if token.lemma_ in action_verbs: #and token.dep_ == "ROOT": tasks.append(sent.text.strip()) break else: tasks.append(sent.text.strip()) except Exception as e: raise Exception(f"Exception in split_tasks_using_spacy >> {str(e)}")
